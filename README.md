# Last Jump : 최후의 도약
## 프로젝트 개요 및 목표
* 장르: 밀리터리 테마의 횡스크롤 자동 러너 (Endless Runner)
* 스타일: 쿠키런 스타일의 간단한 조작 메커니즘을 기반으로, 긴박하고 어두운 분위기의 맵을 배경으로 함.
